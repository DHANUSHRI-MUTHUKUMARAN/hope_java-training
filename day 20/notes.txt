Lifecycle of Thread (state):
    new -> runnalble -> running -> terminated
               ^        ^
                \      /
                block/wait

    -> new - creating new thread
    -> runnalble - waiting queue in CPU before running
    -> running - Thread executing on CPU 
    -> terminated - thread completed 
    -> block/wait state - thread paused
                        - using sleep()
                        - wait()
